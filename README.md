# Shuup PagSeguro

PagSeguro gateway payment add-on for Shuup using *Checkout Transparente*

## Payment Methods
* Boleto
* Online Debit

## Compatibility
* Shuup v0.5.0
* [Tested on Python 3.4 and 3.5](https://travis-ci.org/rockho-team/shuup-pagseguro)
